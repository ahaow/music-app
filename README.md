# music-app

仿照**网易云音乐APP**的移动webapp

api接口地址：[网易云音乐 NodeJS 版 API](https://binaryify.github.io/NeteaseCloudMusicApi/#/)


## 项目结构
	
	- src 主要开发代码
	    -- assets 静态资源
	    -- components 封装组件库
	    -- pages 每一个路由对应的页面
	        -- index  首页
	        -- login  登录页
	    -- reducers.js  reducers入口文件
	    -- rudex 所有redux操作文件夹
	    -- index 入口文件


## 项目依赖

    -- react + react-router
    -- redux + react-redux + redux-thunk 状态管理
    -- axios ajax请求库
    -- react-transition-group 动画
